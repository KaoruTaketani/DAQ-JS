mean
```math
\mu=\frac{\sum _{i=1}^n a_i}{n} 
```

weighted mean
```math
\mu_w = \frac{\sum _{i=1}^n a_i w_i}{\sum _{i=1}^n w_i}
```

standard deviation
```math
\sqrt{\frac{\sum _{i=1}^n |a_i-\mu|^2}{n-1} }
```

weighted standard deviation
```math
\sqrt{\frac{\sum _{i=1}^n w_i|a_i-\mu_w|^2}{\sum _{i=1}^n w_i} }
```
